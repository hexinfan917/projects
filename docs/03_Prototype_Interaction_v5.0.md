| 交互元素 | 交互行为 | 触发效果 |
|---------|---------|---------|
| 搜索框 | 点击 | 跳转搜索页，唤起键盘 |

| 课程卡片 | 点击 | 跳转课程详情页 |
| 页面整体 | 下拉 | 触发下拉刷新，重新加载数据 |
| 页面整体 | 上拉 | 触发上拉加载，加载更多内容 |

#### 2.1.3 数据加载逻辑

```
初始化加载流程：
+-- 并行加载
|   +-- 获取用户定位 -> 获取附近POI
|   +-- 获取轮播图数据
|   +-- 获取热门路线（前5条）
|   +-- 获取直播列表（前4个）
|   +-- 获取课程推荐（前4个）
|
下拉刷新：
+-- 清空当前列表数据
+-- 重新执行初始化加载流程
+-- 刷新完成后收起刷新动画

上拉加载：
+-- 检测当前列表类型
+-- 加载下一页数据（page + 1）
+-- 追加到现有列表
+-- 无更多数据时显示没有更多了
```

#### 2.3.2 安全视频强制观看逻辑

```
用户点击预订按钮时：

IF 用户未观看过安全视频 THEN
    弹出安全视频弹窗
    视频播放结束后显示确认按钮
    用户点击确认后标记为已观看
    跳转订单确认页
ELSE
    直接跳转订单确认页
END IF

安全视频弹窗规格：
- 视频尺寸：16:9
- 视频时长：约60-90秒
- 不可跳过（至少观看80%）
- 底部显示已观看进度条
- 观看完成后确认按钮高亮可点击
```

## 三、关键交互流程

### 3.1 预订流程

```
+-------------+     +-------------+     +-------------+
|   浏览路线   | --> |  选择日期   | --> |  填写信息   |
+-------------+     +-------------+     +-------------+
      |                                         |
      | 点击路线卡片                             | 安全视频
      v                                         v
+-------------+     +-------------+     +-------------+
|  路线详情页  |     |  日期选择器  |     |  观看视频   |
+-------------+     +-------------+     +-------------+
                                                |
                                                v
+-------------+     +-------------+     +-------------+
|  预订成功    | <-- |   支付订单   | <-- |  确认订单   |
+-------------+     +-------------+     +-------------+
      |
      v
+-------------+
|  分享/查看   |
|   订单      |
+-------------+
```

**流程说明：**

1. **浏览路线**：用户在首页或路线列表浏览路线
2. **选择日期**：点击预订后弹出日期选择器，显示可预订日期和价格
3. **安全视频**：首次预订需观看安全须知视频
4. **填写信息**：填写联系人、参与人、宠物信息
5. **确认订单**：选择保险、装备、优惠券，确认价格
6. **支付订单**：调起微信支付
7. **预订成功**：显示成功页面，可分享或查看订单

## 四、状态与异常处理

### 4.1 页面状态

#### 4.1.1 加载状态

| 场景 | 表现形式 |
|-----|---------|
| 页面首次加载 | 全屏骨架屏动画 |
| 列表加载更多 | 底部Loading动画 |
| 表单提交 | 按钮Loading状态 |
| 图片加载 | 灰色占位图 + 加载动画 |

#### 4.1.2 空状态

| 场景 | 图标 | 文案 | 操作按钮 |
|-----|-----|------|---------|
| 无搜索结果 | 搜索空图标 | 没有找到相关内容 | 返回首页 |
| 无订单 | 订单空图标 | 还没有订单哦 | 去逛逛 |
| 无收藏 | 收藏空图标 | 还没有收藏内容 | 去逛逛 |
| 无消息 | 消息空图标 | 暂时没有消息 | 刷新 |
| 网络错误 | 断网图标 | 网络开小差了 | 重试 |

#### 4.1.3 错误状态

```
错误页面通用结构：
+-------------------------------------+
|                                     |
|          [错误图标]                 |
|                                     |
|          错误标题                    |
|          错误描述信息                |
|                                     |
|          [主要操作按钮]              |
|          [次要操作按钮]              |
|                                     |
+-------------------------------------+
```

**常见错误码：**

| 错误码 | 标题 | 描述 | 操作 |
|-------|-----|------|-----|
| 404 | 页面不存在 | 您访问的页面已失效或不存在 | 返回首页 |
| 500 | 服务器繁忙 | 请稍后再试 | 刷新页面 |
| 403 | 无权限访问 | 您没有权限查看此内容 | 返回首页 |
| 401 | 登录已过期 | 请重新登录 | 立即登录 |

#### 4.1.4 网络异常处理

```
网络检测逻辑：
+-- 请求发起前
|   +-- 检测网络状态
|   +-- 无网络：弹出网络提示，阻止请求
|
+-- 请求超时（10秒）
|   +-- 显示重试按钮
|   +-- 记录错误日志
|
+-- 网络恢复
    +-- 自动刷新当前页面
    +-- Toast提示网络已恢复
```

### 4.2 订单状态流转

```
                    +-------------+
                    |   待支付    |
                    |  (15分钟)   |
                    +-------------+
                          |
            +-------------+-------------+
            |                           |
         支付成功                      超时未支付
            |                           |
            v                           v
    +-------------+             +-------------+
    |   已支付    |             |   已取消    |
    |   待出行    |             +-------------+
    +-------------+
            |
    +-------+-------+
    |               |
  出行日期到        用户取消
    |               |
    v               v
+-------------+   +-------------+
|   进行中    |   |  退款中    |
|   待完成    |   |  (审核中)   |
+-------------+   +-------------+
    |                   |
    | 用户确认完成       | 退款完成
    | 或自动完成(7天)    |
    v                   v
+-------------+   +-------------+
|   已完成    |   |  已退款    |
|   待评价    |   +-------------+
+-------------+
    |
    +-- 用户评价 --> 已评价
```

**各状态说明：**

| 状态 | 说明 | 可操作 |
|-----|------|-------|
| 待支付 | 订单创建后15分钟内有效 | 支付、取消 |
| 已支付 | 等待出行日期 | 申请退款 |
| 待出行 | 出行前24小时 | 修改信息、退款 |
| 进行中 | 活动当日 | 联系客服 |
| 已完成 | 活动结束 | 评价、再次预订 |
| 已取消 | 用户主动取消或超时 | 重新预订 |
| 退款中 | 退款申请审核中 | 查看进度 |
| 已退款 | 退款已到账 | 重新预订 |
